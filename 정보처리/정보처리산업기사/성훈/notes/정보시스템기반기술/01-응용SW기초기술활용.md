# 01. 응용SW기초기술활용

> 출처: `_공통/출제기준/정보시스템기반기술.md`의 주요항목 순번을 따른다.

## 운영체제 기초 활용

### 핵심 개념
- **운영체제(OS)**: 컴퓨터 자원(CPU·메모리·디스크)을 관리하고 사용자·응용프로그램 간 인터페이스를 제공하는 시스템 소프트웨어.
- **프로세스·스케줄링**: 프로세스 = 실행 중인 프로그램(PCB 보유). 선점형(RR)·비선점형(FCFS·SJF·HRN) 스케줄링, 세마포어(P/V) 동기화, 교착상태(Deadlock)가 핵심.
- **메모리 관리**: 고정/동적 분할 — 배치 전략(최초·최적·최악 적합), 세그먼테이션, 가상기억장치(페이징), 내적/외적 단편화.
- **파일·디렉터리**: 트리 구조 디렉터리(마스터+사용자별 서브), UNIX 권한(`chmod`), 디스크 스케줄링(FCFS·SCAN), 해싱(동일 버킷 = Synonym).

### 암기 포인트
- ⭐ 스케줄링: **FCFS**(선착순) · **SJF**(최단작업우선) · **HRN** = (대기+서비스)/서비스 · **RR**(시간할당량 커지면 → FCFS화).
- ⭐ 세마포어: **P**(wait, 자원감소·대기) / **V**(signal, 자원증가·깨움). 교착상태 4조건 = **상호배제·점유대기·비선점·순환대기**.
- ⭐ **내적** 단편화 = 분할보다 작아서 남는 공간 / **외적** 단편화 = 분할 사이 빈 공간.
- ⭐ 접근 제어 행렬: **수평(행)** = 권한 리스트(Capability), **수직(열)** = 접근 제어 리스트(ACL).

## 네트워크 기초 활용

### 핵심 개념
- **프로토콜**: 서로 다른 기기 간 데이터 교환을 원활히 하도록 표준화한 통신 규약. OSI 7계층(물리·데이터링크·네트워크·전송·세션·표현·응용)이 기본.
- **TCP/UDP·IP**: TCP = 연결형·신뢰성, UDP = 비연결형·신속. IP 주소 체계(IPv4=32비트, IPv6=128비트).
- **데이터 통신**: LAN 매체(100Base-T=UTP 케이블), 라우팅 프로토콜, 패킷 교환, 비동기/동기 전송, 디지털 코드(해밍 코드=에러 검출·교정).

### 암기 포인트
- ⭐ **IPv6** = 128비트, 유니캐스트·멀티캐스트·**애니캐스트**(브로드캐스트 X), 인증·보안 기능 포함.
- ⭐ OSI: **데이터링크** = 프레이밍·오류제어(HDLC·PPP) / **전송** = 종점 간 연결·오류·흐름제어(TCP·UDP).
- ⭐ 라우팅 프로토콜이 아닌 것 주의(UDP 등이 오답). 100Base-T 전송매체 = **UTP(UTP 케이블)**.
- ⭐ **해밍 코드** = 에러 검출 **및 교정** 가능 / **그레이 코드** = 한 비트만 변화(1010₂ → 1111).

> 현재 기출 문항 중 이 세부항목에 직접 대응하는 것은 없으나, 빌드 자동화 도구·형상관리 관련 문항은 `03. 테스트및배포 > 애플리케이션 배포` 섹션에서 다룬다.
