# 01. 프로그래밍언어활용

> 출처: `_공통/출제기준/프로그래밍언어활용.md`의 주요항목 순번을 따른다.

## 기본문법 활용

### 핵심 개념
- 정의: C·Java·Python 등의 데이터 타입, 변수, 연산자, 제어문(조건문·반복문)의 기본 문법을 이해하고 코드의 실행 결과를 추적하는 능력.
- 핵심 포인트: 연산자 우선순위, 형변환(캐스팅), 포인터 참조, 배열 인덱스, 변수 스코프(지역·전역·외부)의 정확한 이해.
- 자주 헷갈리는 함정: `&&` vs `||` 단축 평가, `%`는 정수 전용, `(int)2.9`는 버림(2), 삼항 연산자 `조건 ? 참 : 거짓`.

### 암기 포인트
- ⭐ 제어문자: `\b`(백스페이스, 커서 좌측 이동), `\t`(탭), `\n`(줄바꿈), `\r`(복귀).
- ⭐ C언어 상수: `const` 예약어, `#define` 전처리. unsigned는 `float` 불가.
- ⭐ 삼항 연산자: `max = (a > b) ? a : b;`
- ⭐ 기억 클래스: `auto`(블록 내 지역), `static`(정적), `extern`(외부 참조), `register`(레지스터).

- 핵심 포인트: 추상 클래스(인스턴스화 불가, 추상 메소드 포함), 캡슐화(정보은닉), 일반화(상속의 역방향, 공통 속성 추출).
- 자주 헷갈리는 함정: 추상 클래스는 "객체 생성 가능"이 틀린 설명. 스크립트 언어 예: Python, JavaScript, Ruby, PHP (컴파일러 언어 아님).

### 암기 포인트

- ⭐ 스크립트 언어: 컴파일 없이 인터프리터 방식, Python·JavaScript·Ruby·PHP·Shell.

## 라이브러리 활용

### 핵심 개념
- 정의: C언어 표준 라이브러리(입출력·수학·문자열), 파일 입출력 함수, Java 예외 처리의 활용.
- 핵심 포인트: `getchar()`(문자 입력), `putchar()`(문자 출력), `fscanf()`/`fread()`(파일 읽기), `math.h`(수학 함수), `stdlib.h`(유틸리티).
- 자주 헷갈리는 함정: 예외 처리는 `try-catch-finally` 구조, 예외를 처리하지 않으면 프로그램 비정상 종료.

### 암기 포인트
- ⭐ C 입출력: `getchar()`/`putchar()`(문자 1개), `scanf()`/`printf()`(서식), `gets()`/`puts()`(문자열).
- ⭐ C 라이브러리: `stdio.h`(표준입출력), `math.h`(수학), `stdlib.h`(동적할당·변환), `string.h`(문자열).
- ⭐ Java 예외: `try`~`catch`~`finally`, `throws`로 위임, `throw`로 발생.
