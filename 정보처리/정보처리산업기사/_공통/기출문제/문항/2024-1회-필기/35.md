```java
if (a > b)
    max = a;
else if (a <= b)
    max = b;
```

① max = (a > b) ? a : b;
② (a > b) ? max = a : max = b;
③ max = (a <= b) ? a : b;
④ (a <= b) ? max = a : max = b;
