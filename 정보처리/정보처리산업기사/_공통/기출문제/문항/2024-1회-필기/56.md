아래 SQL 문에서 WHERE 절의 조건이 의미하는 것은?

```sql
SELECT 이름, 과목, 점수
FROM 학생
WHERE 이름 NOT LIKE ‘박_ _’;
```

① ‘박’으로 시작되는 모든 문자 이름을 검색한다.
② ‘박’으로 시작하지 않는 모든 문자 이름을 검색한다.
③ ‘박’으로 시작하는 3글자의 문자 이름을 검색한다.
④ ‘박’으로 시작하지 않는 3글자의 문자 이름을 검색한다.
