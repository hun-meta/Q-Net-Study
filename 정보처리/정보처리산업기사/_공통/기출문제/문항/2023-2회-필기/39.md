다음 C언어 프로그램이 실행되었을 때, 실행 결과는?

```c
#include <stdio.h>
#include <stdlib.h>
int main(int argc, char* argv[]) {
    int i = 0;
    while (1) {
        if (i == 4) {
            break;
        }
        ++i;
    }
    printf("i = %d", i);
    return 0;
}
```

① i = 0
② i = 1
③ i = 3
④ i = 4
