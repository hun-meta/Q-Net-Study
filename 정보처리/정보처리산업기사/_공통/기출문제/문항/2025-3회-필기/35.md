다음 JAVA 프로그램이 실행되었을 때, 실행 결과는?

```java
public class Main {
  public static void main(String[] args) {
    System.out.print("(a) " + 10/4);
    System.out.print(", (b) " + 10.0/4);
  }
}
```

① (a) 2, (b) 2.5
② (a) 10/4, (b) 10.0/4
③ (a) 2.5, (b) 2.5
④ (a) 2.5, (b) 2
