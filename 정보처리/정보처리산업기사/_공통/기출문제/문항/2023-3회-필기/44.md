다음 트리를 전위 순서(Pre-order)로 운행한 결과는?

```
            A
          /   \
        B       C
       /       / \
      D       E    F
             / \
            G   H
```

- A의 왼쪽 자식 = B, 오른쪽 자식 = C
- B의 왼쪽 자식 = D
- C의 왼쪽 자식 = E, 오른쪽 자식 = F
- E의 왼쪽 자식 = G, 오른쪽 자식 = H

① A B C D E F G H
② D B G H E F C A
③ A B D C E G H F
④ B D G H E F A C
