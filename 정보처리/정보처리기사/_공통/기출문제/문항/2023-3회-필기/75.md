다음은 파이썬으로 만들어진 반복문 코드이다. 이 코드의 결과는?

```python
>> while(True) :
     print('A')
     print('B')
     print('C')
     continue
     print('D')
```

① A, B, C 출력이 반복된다.
② A, B, C
③ A, B, C, D 출력이 반복된다.
④ A, B, C, D 까지만 출력된다.
