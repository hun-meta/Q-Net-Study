다음 C언어 프로그램에서 밑줄 친 부분과 동일한 의미를 가지는 것은 어떤것인가?

```c
#include <stdio.h>
main() {
    int a, b;
    for (a = 0; a < 2; a++)
        for (b = 0; b < 2; b++)
            printf("%d", !a && !b);
}
```

(밑줄 친 부분: `!a && !b`)

① !a || !b
② !(a || b)
③ a && b
④ a || b
