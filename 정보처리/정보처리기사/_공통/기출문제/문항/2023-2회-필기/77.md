다음 자바 프로그램 조건문에 대해 삼항 조건 연산자를 사용하여 옳게 나타낸 것은?

```java
int i = 7, j = 9;
int k;
if (i > j)
    k = i - j;
else
    k = i + j;
```

(선택지는 각각 세 줄 코드이며, 아래에는 `;` 로 구분해 한 줄로 이어 표기)

① int i = 7, j = 9; int k; k = (i > j) ? (i - j) : (i + j);
② int i = 7, j = 9; int k; k = (i < j) ? (i - j) : (i + j);
③ int i = 7, j = 9; int k; k = (i > j) ? (i + j) : (i - j);
④ int i = 7, j = 9; int k; k = (i < j) ? (i + j) : (i - j);
