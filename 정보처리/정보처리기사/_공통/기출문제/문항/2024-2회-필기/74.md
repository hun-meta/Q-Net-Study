다음은 어떤 프로그램 구조를 나타낸다. 모듈 F에서의 Fan-In과 Fan-Out의 수는 얼마인가?

> 프로그램 구조도(모듈 간 호출 관계):
>
> ```
>             A
>          ／  |  ＼
>         B    C    D
>       ／  ＼  |  ／
>      E       F
>           ／   ＼
>          G       H
> ```
>
> - A → B, A → C, A → D
> - B → E, B → F, C → F, D → F
> - F → G, F → H

① Fan-In : 2, Fan-Out : 3
② Fan-In : 3, Fan-Out : 2
③ Fan-In : 1, Fan-Out : 2
④ Fan-In : 2, Fan-Out : 1
