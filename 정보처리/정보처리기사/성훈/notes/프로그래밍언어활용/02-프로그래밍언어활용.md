# 02. 프로그래밍언어활용

> 출처: `_공통/출제기준/프로그래밍언어활용.md`의 주요항목 순번을 따른다.

## 기본문법 활용

### 핵심 개념
- 정의: 프로그래밍 언어의 **데이터 타입·변수·연산자·제어문·포인터** 등 기본 문법을 활용하는 능력. 기출 최다 출제 세부항목.
- 연산자 분류: 산술(`+ - * / %`)·관계(`> < ==`)·논리(`&& || !`)·**시프트(`<< >>`, 산술 아님)**·증감(`++ --`)·대입(`=`, 산술 아님)·**삼항(`조건 ? 참 : 거짓`)**.
- 우선순위(높→낮): `()` > 단항·증감 > 산술(`/` 등) > 시프트(`<<`) > 관계(`< ==`) > 논리(`||`). 드모르간 법칙: `!a && !b ≡ !(a||b)`.
- 변수명 규칙: 영문·숫자·밑줄(`_`), 숫자로 시작 불가, **예약어(short·int 등)·공백·하이픈(`-`) 사용 불가**.
- 포인터(C): `&`(주소), `*`(역참조). Java는 `while(y--)`처럼 int→boolean 자동변환 안 함(**컴파일 에러**).
- 데이터 타입(Java): `char`=2Byte, `float`/`double`=실수형(`short`·`byte`는 정수형).

## 언어특성 활용

### 핵심 개념
- 정의: 언어를 **절차적·객체지향·스크립트·선언형**으로 분류하고, 각 언어의 특성을 이해한다.
- 스크립트 언어: **Python·JavaScript·PHP·ASP·Ruby·Perl**. **아닌 것**: Fortran·Cobol·C(절차적·컴파일 언어).
- 객체지향(OOP): 캡슐화·상속·다형성·추상화. **Garbage Collector**(Java)는 참조를 잃은 객체 자동 제거. 오버로딩(동명 다인자) vs 오버라이딩(재정의).
- 함정: 언어 선정 시 '컴파일러 독창성'은 고려사항이 **아님**.

## 라이브러리 활용

### 핵심 개념
- 정의: **라이브러리**(재사용 가능한 함수·클래스 모음)·표준함수·입출력·예외처리를 활용하는 능력.
- C 표준라이브러리: `string.h`(strlen·strcpy·strcmp·strcat·strrev), `stdlib.h`(atoi/strtol 등 문자열↔수치 변환), `stdio.h`(printf·scanf). `scanf("%d", &n)` — 형식지정자 + **주소연산자 `&`**.
- Java 입출력: `print`·`println`·`printf` (존재), **`printing`·`prn` (미존재, 함정)**. 제어문자 `\t`=탭.
- 예외 처리: `try-catch-finally`, `throw`/`throws`. Python `print(sep=, end=)` 옵션. import로 외부 패키지(`java.util.Random`) 사용.
