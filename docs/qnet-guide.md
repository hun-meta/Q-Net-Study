# Q-Net 국가자격 가이드 (조사 자료)

> 2026-07-20 조사. 출제기준·일정은 매년 바뀌므로 실제 응시 전 Q-Net 원문을 반드시 재확인할 것.
> 공식 사이트: https://www.q-net.or.kr (종목 상세: crf005.do, 출제기준 자료실: cst006.do)

## 1. 자격 등급 체계 (이 저장소의 `{종류}` 디렉토리 기준)

위계: **기술사 > 기능장 > 기사 > 산업기사 > 기능사** + 별도 카테고리 **전문자격**(국가전문자격)

| 등급 | 성격 | 응시자격 요약 |
|------|------|--------------|
| 기능사 | 입문 등급 | **제한 없음** — 누구나 응시 |
| 산업기사 | 중급 실무 | 기능사+실무 1년 / 관련학과 전문대졸 / 실무 2년 등 |
| 기사 | 중견 기술 인력 | 산업기사+1년 / 기능사+3년 / 관련학과 4년제 대졸(예정) / 실무 4년 등 |
| 기능장 | 기능계 최고 등급 | 산업기사 이상+실무 5년 / 기능사+7년 / 실무 9년 등 |
| 기술사 | 기술계 최고 등급 | 기사+실무 4년 / 산업기사+5년 / 실무 9년 등 |
| 전문자격 | 개별법 기반 (공인중개사, 세무사, 공인노무사 등 37종) | 종목별 상이 |

- 국가기술자격 종목 수: 약 490종 (2025년 기준, 참고치)
- 직무분야 26개 대분류: 정보통신, 전기·전자, 안전관리, 건설, 기계, 화학, 환경·에너지, 음식서비스, 경영·회계·사무 등
- 응시 가능 여부는 Q-Net 로그인 후 "응시자격 자가진단"으로 확인 가능

## 2. 시험 구조와 합격 기준

| 구분 | 형태 | 합격 기준 |
|------|------|----------|
| 필기 | 객관식 4지선다 (대부분 CBT) | **과목당 40점 이상 + 전 과목 평균 60점 이상** (한 과목 40 미만 = 과락) |
| 실기 | 필답형 / 작업형 / 혼합 (필답형 CBT 확대 중) | **60점 이상** |

- 필기 합격 후 **2년간** 실기 응시 가능 (필기 면제 기간)
- 필기 합격 후 응시자격 서류 미제출 시 합격 무효 (산업기사 이상)

## 3. 출제기준 계층 구조 ← 노트 목차의 기준

출제기준 문서(Q-Net 자료실에서 종목별 `.hwp` 배포)는 4단계 트리다:

```
과목 (예: 소프트웨어 설계)
└─ 주요항목 (예: 요구사항 확인)        ← 노트 파일 1개 단위
   └─ 세부항목 (예: 요구사항 확인)      ← 노트 안의 ## 섹션
      └─ 세세항목 (예: UML, 애자일)     ← 섹션 안의 키워드/소제목
```

- 출제기준은 통상 **3년 주기**로 개편된다. 노트 작성 전 적용 기간(예: 2024.1.1~2026.12.31)을 확인하고 `_공통/info.md`에 기록할 것.

## 4. 시험 일정 체계

- **정기시험** (기사·산업기사 등): 연 **3회** — 회차마다 `필기접수 → 필기 → 발표 → 서류제출 → 실기 → 발표` 사이클
- **상시시험**: 기능사 등 일부 종목은 연중 수시 (CBT)
- 연간 일정: Q-Net > 시험일정 (crf021.do)
